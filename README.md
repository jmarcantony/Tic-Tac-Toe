# Tic-Tac-Toe
Tic Tac Toe game ❌⭕

# Instalation on Windows
* `git clone https://github.com/ninjahacker123/Tic-Tac-Toe.git`
* `cd Tic-Tac-Toe`
* `python main.py`

# Installation on Linux and Unix like OS
* `git clone https://github.com/ninjahacker123/Tic-Tac-Toe.git`
* `cd Tic-Tac-Toe`
* `python3 main.py`
